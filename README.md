# Productivity Time Tracker – Chrome Extension

## Project Overview

Productivity Time Tracker is a browser-based productivity monitoring application developed as a Chrome Extension to help users understand and improve their online browsing habits. The extension automatically tracks the amount of time spent on different websites and categorizes them into **Productive**, **Unproductive**, and **Neutral** groups. The collected browsing information is synchronized with a backend server and presented through an interactive dashboard that provides meaningful productivity analytics and website usage insights.

The primary objective of this project is to demonstrate the implementation of browser extension development using modern web technologies. The application combines Chrome Extension APIs, backend services, RESTful API communication, and data visualization techniques to create a complete productivity monitoring system. The extension operates in the background without interrupting the user's browsing experience and continuously records browsing activity to generate accurate productivity reports.

This project was developed using **Cursor AI**, which assisted in generating the project structure, backend logic, extension components, and dashboard interface. The generated code was further configured, tested, and refined to ensure proper functionality and seamless integration between the frontend, backend, and browser extension.

## Working Process

The Productivity Time Tracker operates as a background Chrome Extension that continuously monitors the active browser tab. Whenever the user visits a website, the extension records the time spent on that particular domain. Each website is compared against predefined productivity categories and classified as **Productive**, **Unproductive**, or **Neutral** based on its domain.

The collected browsing data is periodically synchronized with a backend server developed using Node.js and Express.js. The backend processes the received information and stores it in a local JSON database. The dashboard retrieves this processed data through REST APIs and displays productivity statistics, category-wise browsing time, total tracked time, productivity score, weekly reports, and the most frequently visited websites. This workflow provides users with a comprehensive overview of their browsing habits and enables them to evaluate and improve their productivity effectively.
